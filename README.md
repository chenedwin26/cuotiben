# 全科错题本 · GitHub Pages 下载页

访问地址：https://chenedwin26.github.io/cuotiben/

## 🚀 部署步骤

### 第1步：创建 GitHub 仓库
1. 打开 https://github.com/new
2. Repository name 填：`cuotiben`
3. 选择 Public
4. 不要勾选任何初始化选项
5. 点 Create repository

### 第2步：推送文件（在本文件夹执行）
```bash
git init
git add .
git commit -m "全科错题本下载页"
git branch -M main
git remote add origin https://github.com/chenedwin26/cuotiben.git
git push -u origin main
```

### 第3步：开启 Pages + 上传 APK
1. 仓库 Settings → Pages → Source 选 `main` → Save
2. 等1分钟，访问 https://chenedwin26.github.io/cuotiben/ 检查
3. Releases → Create a new release → Tag `v2.0.0` → 上传 `全科错题本.apk` → Publish

## 📱 使用
邀请好友：`https://chenedwin26.github.io/cuotiben/?code=你的邀请码`

页面会自动检测手机系统，展示对应下载按钮和邀请码。
